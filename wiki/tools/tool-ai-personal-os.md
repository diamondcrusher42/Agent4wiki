# AI Personal OS Skills

Onboarding (creates CLAUDE.md, [[concept-soul-md|SOUL.md]], user-profile.md, goals). Daily-log (energy/mood/focus 1-5 + timestamps + auto pattern analysis every 3rd entry). Atomize (Zettelkasten extraction: one idea per note, rewritten in user voice, wikilinked). Cos-review (system health 0-25 score, repetition detection → new skill candidates, ONE highest-leverage action, 30-second status check).

Skill progression pattern: onboarding → logging → knowledge → review → output → ongoing review. Maps to architecture phased rollout: foundation first, then orchestration, then intelligence.
